随机游走的常返性：醉汉与醉鸟能否归家？

发布时间: 2026-05-06 | 分类: 概率论

关键词：随机游走、常返性、随机级数、扩散过程

前言

几天前的凌晨，某人在b站上刷到了一个关于随机游走的视频，但他忘了常返性怎么证明，而视频里也没有提到，遂作此。

想象这样一个场景：一只鸟在三维天空中随机飞行，它能否回到原点？二维平面上的醉汉最终能否找到回家的路？这些问题看似直观，却蕴含着概率论中一个优雅而深刻的结论——拥有更多的选择也意味着更容易迷茫。。

本文将从数学推导出发，系统分析一维、二维和三维对称随机游走的常返性。

1. 引言

设 \(\{S_n\}_{n\ge 0}\) 是定义在 \(\mathbb{Z}^d\) 上的简单对称随机游走，即：

  • \(S_0 = 0\)
  • 对每个 \(n\ge 1\)，\(S_n = X_1 + \cdots + X_n\)，其中 \(\{X_i\}_{i\ge 1}\) i.i.d.
  • \(P(X_i = e_j) = \frac{1}{2d}\)，其中 \(e_j\) 是 \(\mathbb{Z}^d\) 中的单位坐标向量，\(j = 1,\dots,d\)

本文要回答的核心问题是：

随机游走是否以概率 1 无限次返回到原点？

该性质称为常返性（recurrence）；反之称为暂态性（transience）。1921 年，Pólya 给出了完整的维数判别：

  • \(d = 1, 2\)：常返
  • \(d \ge 3\)：暂态

2. 常返性的判别准则

设 \(u_n = P(S_n = 0)\) 表示第 \(n\) 步时回到原点的概率。定义示性变量 \(I_n = \mathbf{1}_{\{S_n = 0\}}\)，则回到原点的总次数为 \(N = \sum_{n=0}^\infty I_n\)。

由 Markov 性可得一个关键判别法：

引理（期望与常返的关系）：\(\mathbb{E}[N] = \sum_{n=0}^\infty u_n\)

  • 若 \(\sum_{n=0}^\infty u_n = \infty\)，则 \(P(\text{无限次返回}) = 1\)（常返）
  • 若 \(\sum_{n=0}^\infty u_n < \infty\)，则 \(P(\text{无限次返回}) = 0\)，且 \(\mathbb{E}[N] < \infty\)（暂态）

因此，问题归结为估计 \(u_n\) 的渐近行为。

3. 一维情形 (\(d=1\))

在 \(\mathbb{Z}\) 上，每一步 \(\Delta S = \pm 1\) 等概率。经过 \(n\) 步后回到原点要求 \(n\) 为偶数且正负步数相等：

\[ u_{2n} = \binom{2n}{n} \left(\frac12\right)^{2n},\quad u_{2n+1} = 0 \]

使用 Stirling 公式：\(n! \sim \sqrt{2\pi n}\left(\frac{n}{e}\right)^n\)，可得

\[ u_{2n} \sim \frac{1}{\sqrt{\pi n}} \]

于是 \(\sum_{n=0}^\infty u_{2n} \sim \sum_{n=1}^\infty \frac{1}{\sqrt{\pi n}} = \infty\)

因此一维简单随机游走常返。

4. 二维情形 (\(d=2\))

在 \(\mathbb{Z}^2\) 上，每一步从四个方向均匀选取：\((1,0), (-1,0), (0,1), (0,-1)\)。

在第 \(n\) 步回到原点要求 \(X\) 方向净位移为 0 且 \(Y\) 方向净位移为 0，且 \(n\) 为偶数。

使用二项式恒等式：\[ u_{2m} = \left( \frac{1}{4^{2m}} \binom{2m}{m} \right)^2 \]

因为两个坐标独立，且每个坐标的一维返回概率如上。因此：

\[ u_{2m} \sim \frac{1}{\pi m} \]

二维随机游走常返。

5. 三维情形 (\(d=3\))

在 \(\mathbb{Z}^3\) 上，每一步有 6 个方向：\((\pm1,0,0), (0,\pm1,0), (0,0,\pm1)\)。

回到原点要求三个坐标净位移均为 0。步数 \(n = 2m\) 为偶数。

由多维局部极限定理：

\[ u_n \sim \left( \frac{d}{2\pi n} \right)^{d/2} \]

验证：

  • \(d=1\)：\(u_n \sim \left( \frac{1}{2\pi n} \right)^{1/2}\)
  • \(d=2\)：\(u_n \sim \frac{1}{2\pi n}\)
  • \(d=3\)：\(u_n \sim \left( \frac{3}{2\pi n} \right)^{3/2} = O(n^{-3/2})\)

此时：

\[ \sum_{n=1}^\infty u_n \sim C \sum_{n=1}^\infty n^{-3/2} < \infty \]

因为 \(3/2 > 1\)。因此三维随机游走暂态。

6. 严格证明的轮廓（Pólya 定理）

更严谨的证明需要：

  1. 用 Fourier 分析写出返回概率：\[ u_n = \frac{1}{(2\pi)^d} \int_{[-\pi,\pi]^d} \left( \frac{1}{d} \sum_{j=1}^d \cos \theta_j \right)^n d\theta_1\cdots d\theta_d \]
  2. 分析积分在 \(\theta=0\) 附近的渐近：\[ \frac{1}{d} \sum \cos \theta_j \approx 1 - \frac{|\theta|^2}{2d} \]
  3. 于是积分渐近于 \[ \int_{\mathbb{R}^d} e^{-n|\theta|^2/(2d)} d\theta \sim (2\pi d/n)^{d/2} \]
  4. 因此 \(u_n \approx C_d n^{-d/2}\)
  5. 级数 \(\sum n^{-d/2}\) 收敛当且仅当 \(d/2 > 1\)，即 \(d > 2\)

7. 直观解释与物理类比

直观理解：

  • \(n\) 步后均方根位移 \(\sim \sqrt{n}\)
  • 可达区域的"体积"在 \(d\) 维中是 \(O(n^{d/2})\)
  • 随机游走的"足迹"数（不同位置的数量）\(\sim n\)
  • 若 \(n^{d/2}\) 增长快于 \(n\)（即 \(d > 2\)），则足迹会越来越稀疏，新的空间永远存在，无须返回原点
  • 临界情况 \(d=2\) 时，足迹密度趋于常数，但实际证明仍是常返

8. 结论

维度\(u_n\) 渐近\(\sum u_n\)常返性
1\(n^{-1/2}\)发散常返
2\(n^{-1}\)发散常返
3+\(n^{-d/2}\)收敛（若 \(d>2\)）暂态

这一结果展示了随机过程中维数的关键作用：维数 2 是常返性的临界点。在数学上，它来源于随机游走协方差矩阵的特征值；在物理上，它与扩散方程的基本解在二维和三维的不同衰减有关。
